Multiples of 3 and 7 
Find the sum of all the positive multiples of 3 or 7 below a given positive integer 'n'.

import sys 
import math 
n = int(sys.argv[1]) 
total = 0
for i in range(0,n+1):
    if (i % 3 == 0 or i % 7 == 0):
        total = total + i
print(total)
